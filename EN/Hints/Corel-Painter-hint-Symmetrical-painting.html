對稱繪圖

「鏡射繪製」模式可讓您建立對稱繪圖。

• 當您按一下「鏡射繪製」工具 時，繪圖視窗中會顯示一道平面。這讓您只需繪製物件的一半，Corel Painter 便可透過重製筆刷筆觸，自動在物件的另一邊複製出鏡射影像。
• 若要建立鏡射繪製，請從「筆刷選取器」選擇筆刷類別與筆刷變體，然後在鏡射平面的任一側進行繪製。
• 若要調整鏡射平面的對齊方式，請按一下「鏡射繪製」工具 ，然後按一下屬性列上的「垂直平面」 或「水平平面」 按鈕。
• 若要移動鏡射平面，將滑鼠暫留在鏡射平面中間的節點，然後按一下並拖曳至新的位置。
• 若要旋轉鏡射平面，將滑鼠暫留在鏡射平面，然後按一下並拖曳至新的角度。
• 若要在繪圖時隱藏鏡射平面，請按 Command + Shift + M (macOS) 或 Ctrl + Shift + M (Windows)。
• 若要停用鏡射繪製模式，請按 Command + Option + M (macOS) 或 Ctrl + Alt + M (Windows)。

如需「鏡射繪製模式」的詳細資訊，請參閱「說明」。
