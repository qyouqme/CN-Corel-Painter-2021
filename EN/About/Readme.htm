系統需求

Windows:

  • Windows 10* (64 位元) 或 Windows 7 (64 位元) (含最新更新)
  • * Corel Painter 2021 支援 Windows 10 的 1909 版，以及於 Corel Painter 2021 生命週期中推出的更新版本。
  • Intel Core 2 Duo 或 AMD Athlon 64 X2 處理器，時脈
    • 4 實體核心/8 個邏輯核心或更高等級 (建議)
    • 支持AVX2指令集（推薦）
  • 具有OpenCL（1.2或更高版本）兼容性的現代GPU
  • 4 GB RAM
    • 8 GB RAM 或更多 (建議使用)
  • 1.2 GB 硬碟空間供應用程式檔案使用*
    • 固態磁碟機 (建議使用)
  • 1280 x 800 @100% (或更新版本) 螢幕解析度
    • 1920 x 1200 @ 150% (或更新版本) (建議使用)
  • 鼠標或兼容Wintab的平板電腦
  • Microsoft Internet Explorer 11 或更新版本 (含最新更新)
  • 若要啟用產品，您必須先連線至網際網路並註冊。

可能需要額外空間供 Microsoft.NET Framework 4.7 使用

Mac:

  • macOS 10.15 和 10.14, (含最新修訂)
  • Intel Core 2 Duo
    • 4 實體核心/8 個邏輯核心或更高等級 (建議)
  • 具有OpenCL（1.2或更高版本）兼容性的現代GPU（推薦）)
  • 4 GB RAM
    • 8 GB RAM 或更多 (建議使用)
  • 1.8 GB 硬碟空間供應用程式檔案使用
    • 固態磁碟機 (建議使用)
    • 不支持區分大小寫的文件系統。
  • 1280 x 800 @ 100% (或更新版本) 螢幕解析度
    • 1920 x 1200 @ 150% (或更新版本) (建議使用)
  • 滑鼠或繪圖板
  • 若要啟用產品，您必須先連線至網際網路並註冊。

安裝

安裝 Corel Painter 2021 之前

  • 確認您的系統日期/時間已正確設定。
  • 關閉所有應用程式，包含所有病毒偵測程式以及在 Dock、系統匣或 Windows 工作列中開啟的應用程式。如果不關閉，可能會增加安裝時間並干擾安裝過程。
  • 確認在您要安裝應用程式的磁碟機具有足夠的可用磁碟空間。

網路/工作站

  • 網路管理員可以參考《網路部署指南》，該指南專供購買多份 Corel Painter 2021 授權的機構使用。如需詳細資訊，請聯絡 Corel 的授權部門或客戶服務中心。

安裝 Corel Painter 2021

  • 將 Corel Painter 2021 光碟片插入 DVD 光碟機。
    (Windows) 如果未顯示「自動執行」畫面，請按一下 Windows 工作列上的「開始」>「執行」，然後輸入 X:\autorun.exe 或 X:\Setup\Setup.exe (其中 X:\ 是 DVD 光碟機).
    (Mac OS) 按兩下套件即可開始安裝。
  • 依照指示安裝應用程式。

解除安裝 Corel Painter 2021

解除安裝 Corel Painter 2021 (Windows 10)

  1. 在「開始」畫面上按住 Corel Painter 2021，或者以滑鼠右鍵按一下 Corel Painter 2021。
  2. 輕觸或按一下「解除安裝」以開啟「控制台」的「程式和功能」頁面。
  3. 選取清單中的「Corel Painter 2021」，然後選擇「解除安裝/變更」。 
  4. 在安裝精靈中，啟用「移除」選項。
    若要保留您的使用者檔案 (如：偏好設定、預設、使用者建立的檔案、自訂檔案)，請取消勾選「移除使用者檔案」核取方塊。

解除安裝 Corel Painter 2021 (Mac)

  1. 將 Corel Painter 2021 啟動程式拖曳至 Dock 上的「垃圾桶」圖示，即可加以移除。
  2. 將 Corel Painter 2021 啟動程式拖曳至 Dock 上的「垃圾桶」圖示，然後依序進行以下步驟：「移除應用程式」>「Corel Painter 2021」資料夾。
  3. 移除使用者：將使用者拖曳至 Dock 上的「垃圾桶」圖示，然後依序進行以下步驟：「程式庫」>「應用程式支援」>「Corel」>「com.corel.Painter2021」，然後移至「Painter 2021」與「Painter 2021 Config」資料夾。
  4. 將使用者拖曳至 Dock 上的「垃圾桶」圖示，然後依序進行以下步驟：「使用者」>「應程式庫」>「偏好設定」>「com.corel.Painter2021」資料夾，然後移至「com.corel.Painter2021.plist」。
  5. 從 [Finder] 選單中選擇 [清空垃圾桶].

試用版、學術版和 OEM 版本

試用版、學術版和 OEM 版中並未提供所有元件。未提供的元件可能包含 Corel 應用程式、功能、協力廠商公用程式與額外的內容檔案。不過，這些元件中，有一些可能仍會在應用程式及文件中提及。

技術支援

某些版本的 Corel 產品 (包括 OEM、「組裝電腦」、光碟片外盒 (僅限 CD)、試用版及學術版)，無法獲得技術支援。

不過，Corel 產品的所有版本都可以使用下列網頁式支援的資源:

  • Corel 支援網站 (www.corel.com/support) - 安裝中心、學習中心等等
  • Corel 知識庫 (www.corel.com/knowledgebase) - 已知問題和常見問題集的可搜尋資料庫資料庫
